# Skill: ceo-briefing

Generate the Weekly Business and Accounting Audit CEO Briefing.
Aggregates data from Odoo accounting, vault activity, and social media stats into one executive summary.

## Trigger

Use this skill when:
- User says "generate CEO briefing", "weekly briefing", "Monday briefing"
- It's Monday morning (scheduled via cron)
- User asks for a business summary or audit

## What to Include

### 1. Accounting Summary (from Odoo MCP)
Use the `odoo-accounting` skill / MCP tools:
- `get_financial_summary(period="this_month")` — revenue, expenses, net profit
- `get_financial_summary(period="last_month")` — comparison
- `list_invoices(status="posted")` — open invoices
- `list_invoices(status="paid")` — recently paid
- Focus on: overdue invoices, outstanding amounts, month-over-month trend

### 2. Vault Activity Summary
Read the vault folders and logs:
- Count items in `/Needs_Action`, `/Pending_Approval`, `/Done` (past 7 days)
- Read `AI_Employee_Vault/Logs/` for recent JSONL entries
- Summarize: emails processed, posts published, approvals pending

### 3. Social Media Summary
Scan `/Done/` for:
- `DONE_*_PUBLISHED_LINKEDIN_POST_*.md` — LinkedIn posts this week
- `DONE_*_PUBLISHED_FACEBOOK_POST_*.md` — Facebook posts this week
- Count and summarize what was posted

### 4. Action Items
List items currently in `/Pending_Approval` — what needs the CEO's attention.
List any overdue invoices from Odoo.

### 5. Anomalies / Alerts
- Any invoices overdue > 30 days
- Unexpected high expenses
- Approval requests older than 48h

## Output Format

Save to: `AI_Employee_Vault/Done/CEO_BRIEFING_<YYYY-MM-DD>.md`

```markdown
# CEO Briefing — Week of <date>
*Generated by AI Employee on <timestamp>*

---

## Financial Snapshot
| Metric | This Month | Last Month |
|--------|------------|------------|
| Revenue Invoiced | $X | $X |
| Revenue Collected | $X | $X |
| Expenses | $X | $X |
| Net Profit | $X | $X |
| Outstanding Invoices | $X (N invoices) | — |

### Overdue Invoices
<list or "None">

---

## Operations Summary (Past 7 Days)
- Emails processed: N
- LinkedIn posts published: N
- Facebook posts published: N
- Approvals completed: N
- Approvals still pending: N

---

## Pending Action Items (Requires CEO)
<list files in /Pending_Approval with age>

---

## Alerts
<any anomalies>

---
*Next briefing: <next Monday>*
```

## HITL Rules

- **Autonomous**: Reading all data, generating and saving the briefing file
- **No external actions**: This skill only reads and writes to the vault

## Scheduling

To run every Monday at 9am, use the schedule-task skill:
`/schedule-task "ceo-briefing" "0 9 * * 1"`

Or run on-demand: just say "generate CEO briefing"
